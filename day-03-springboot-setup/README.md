# Day 3 — Spring Boot Project Setup

This folder documents the initial setup of a **Spring Boot application**, marking the transition from core Java fundamentals to framework-based backend development. The focus is on understanding project structure, configuration, and application startup rather than business logic.

---

## Objectives

- Create a Spring Boot project using Spring Initializr
- Understand the default project structure
- Run a Spring Boot application successfully
- Learn where configuration and application entry points live

---

## Tech Stack

- Java
- Spring Boot
- Maven
- Embedded Tomcat
